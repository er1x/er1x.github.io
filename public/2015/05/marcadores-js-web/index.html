frontend, nodejs

  • Documentación oficial. Para mí el mejor libro de Backbone.
  • Libro de Backbone. Si te interesa algo más denso y ordenado este es un buen libro gratuito (en inglés).

AngularJS

AngularJS es un framework JS full-stack mantenido por Google.

  • Docu oficial
  • Comparativa de starters de AngularJS

Visualización de Datos

Frecuentemente me surge la necesidad de agregar gráficas a mis proyectos. Normalmente utilizo alguna de estas 3 alternativas:

  • Chartjs. Pocos tipos de gráficas, pero sencillos y bonitos.

  • Highcharts. Una colección enorme de gráficas con muchas opciones de personalización.

  • D3JS. Lo mejor para crear tus propios gráficos. Si necesitas algo muy particular o mucho control sobre lo que estás haciendo, ésta es la opción a tomar. Eso sí, hay que currárselo mucho más que con las otras dos.

Diseño y UX

Aunque el diseño no es mi foco, siempre toca hacer alguna cosilla :)

  • Logopond. Inspiración para logos.
  • FlatUIColors. Colores chulos para cualquier aplicación. Si no te quieres romper la cabeza buscando esquemas de colores.
  • RocketDesign. Aprender a combinar colores.
  • Adobe Color : más colores. En esta web encontrarás esquemas de colores a porrillo, además de poder crear los tuyos propios.
  • SubtlePatterns. Imágenes con patrones para usar de background en webs.
  • Best Designs. Inspiración para diseñar una web.
  • Viewport Sizes. Base de datos con resoluciones de pantalla de un montón de dispositivos.
  • Invision : una plataforma colaborativa para hacer diseños.
  • Canva. Herramienta web genial para crear imágenes sin tener ni idea :P
  • Pttrns. Diseños de aplicaciones para móvil**.

Iconos y Fuentes

Dicen que el 90% del diseño es tipografía. No sé si llega a tanto, pero siempre viene bien contar con fuentes chulas.

  • TypeConnection. Un juego para aprender a combinar tipografías.
  • Typ.io. Otra web de combinación de tipografías
  • FlatIcon. Una gran base de datos con iconos en SVG y otros formatos.
  • IcoMoon. Aplicación web genial para crear nuestro propia fuente de iconos eligiendo los que necesitemos.
  • Font Squirrel
  • Google Fonts
  • What The Font. Si quieres identificar una fuente, esta web es tu amiga. Sube una imagen e intentará identificar de qué fuente se trata.

  • Typekit
